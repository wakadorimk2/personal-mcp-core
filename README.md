# personal-mcp-core

PoE2 などのゲームプレイログを手元に蓄積・検索するための個人用ツールです。
MCP は将来の統合レイヤの候補ですが、いまは CLI で動きます。

## これは何か

- ゲームログ（PoE2）の記録・検索・集計を行う「相棒の核」
- AI が自分を誤解しにくくするための前提・原則を構造として持つ個人基盤
- 完成を目指さず、ゆっくり育てていく Python リポジトリ

MCP はこのリポジトリの本体ではなく、将来の「出口」の一つです。

## 何ができるか

```bash
# PoE2 プレイログを記録する
python -m personal_mcp.server poe2-log-add "アトラスの外縁でボス撃破" --kind=milestone --tags=atlas,boss

# 直近のログを一覧する
python -m personal_mcp.server poe2-log-list --n 10

# タグ・期間で絞り込む
python -m personal_mcp.server poe2-log-list --tag atlas --since 2026-03-01
```

ログは `data/poe2/logs.jsonl` に追記されます。
**注意**: ログに個人情報や機密情報を含めないよう気をつけてください。

## 構成

- `AI_GUIDE.md` — AI が守るべき態度・優先順位・禁止事項
- `CLAUDE.md` — Claude Code 向けの作業ガイド（ファイル構造・規約）
- `src/personal_mcp/` — ゲームログの記録・検索ロジックとアダプタ層
- `docs/` — アダプタ設計・ツール設計・データフロー（必要になったら追加）
- `data/` — ログデータ置き場（必要になったら追加）

## どこへ向かうか

1. **いまここ** — 手動でログを記録・検索できる（CLI）
2. **次** — ゲームクライアントのログを監視して自動記録
3. **その次** — 蓄積データの集計・サマリ生成
4. **将来** — 必要と判断したら MCP / HTTP で AI ツールに接続

このリポジトリは意図的に未完成です。考えが変わったときに書き換えながら、ゆっくり育てていきます。
